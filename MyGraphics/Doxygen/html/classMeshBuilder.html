lengthX- x-axis should start at -lengthX / 2 and end at lengthX / 2
lengthY- y-axis should start at -lengthY / 2 and end at lengthY / 2
lengthZ- z-axis should start at -lengthZ / 2 and end at lengthZ / 2
Returns
Pointer to mesh storing VBO/IBO of reference axes
